<h3>Usage</h3>
<p>Build and run command:</p>
<pre>
    mkdir build && cd build
    cmake ..
    make && ./Sudoku_EXEC
</pre>
<p>Unit tests:</p>
<pre>
    ./Sudoku_UT
</pre>
